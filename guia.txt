## Prerequisitos

## Opción A — Evaluar el algoritmo puro (con detecciones FRCNN)
> Sirve para validar que el tracker Rust es correcto respecto al Python.

### Paso 1 — Ejecutar el tracker Rust dentro del contenedor
```bash
docker exec 2df0dfa78dc4 ./target/release/eval_mot17
```

### Paso 2 — Copiar resultados al host
```bash
docker cp 2df0dfa78dc4:/app/rust_results/. \
    /home/catec/TrackTrack/outputs/rust_results_v2/
```

### Paso 3 — Evaluar
```bash
cd /home/catec/TrackTrack/3.\ Tracker
python3 run_eval.py
```

## Rutas importantes

| Qué | Ruta |
|-----|------|
| Código Rust | `/app/src/` (dentro del contenedor) |
| Resultados Rust FRCNN | `/app/rust_results/` (contenedor) → `outputs/rust_results_v2/` (host) |
| Resultados Rust YOLO | `/app/TrackTrack/outputs/rust_results/data/` (contenedor) → `outputs/rust_results_v2/` (host) |
| Resultados Python | `/home/catec/TrackTrack/outputs/python_results/` (host) |
| Script de evaluación | `/home/catec/TrackTrack/3. Tracker/run_eval.py` |
